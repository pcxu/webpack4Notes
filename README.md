# 食用方式
webpack.config.js，罗列了项目搭建、开发、打包过程中需要使用的基础api及注释，帮忙理解和搭建工程化前端项目。  
`cnpm i --save-dev` 下载依赖  
`npm run dev` 运行项目  
`npm run analyz` 项目打包（访问控制台提示地址，可查询打包内容和引入依赖，有内存大小提示，给予优化思路）。

# 编写意义
熟悉webpack的api，初版希望能脱离脚手架搭建可开发可上线的项目，后续深入了解，进行特定性能优化或其他高级配置。
